# 🧦 Mue - Dobby Widget

welcome to my personal project, created on boring nights.

##Features:
- real time lock
- current date in Portuguese(at the moment)
- System CPU and RAM monitoring
- Dobby in pixel art with animations -breaking, blinking ears wagging)
- phrases that change according to the CPU state

##Technologies:
- [Electron](https://www.electronjs.org/) — to create the window on the desktop
- [systeminformation](https://systeminformation.io/) — to real read data from the system
- HTML, CSS e JavaScript for the look

##How to run locally:
- clone the repository -
```
   git clone https://github.com/seu-usuario/mue.git
   cd mue
```
- install the dependencies: 
```
   npm install
```
- launch the widget: 
```
   npm start
```
